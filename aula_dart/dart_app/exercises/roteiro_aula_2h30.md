# Roteiro de Aula Dart + POO (2h30)

Este roteiro usa somente Dart no console.
Todos os exemplos foram pensados para iniciantes.

## Cronograma sugerido

1. 00:00-00:15 - Fundamentos do programa Dart
- Estrutura basica
- Variaveis e tipos
- Interpolacao de strings

2. 00:15-00:40 - Controle de fluxo e repeticao
- if/else
- switch
- for, while, for-in

3. 00:40-01:00 - Funcoes, listas e mapas
- funcoes simples, parametros, retorno, arrow
- listas e mapas

4. 01:00-01:40 - POO basica
- classe simples
- objetos
- metodos
- construtores e construtores nomeados
- encapsulamento

5. 01:40-02:05 - POO avancada
- heranca
- override
- classes abstratas
- interfaces com implements
- mixins
- membros estaticos

6. 02:05-02:30 - Excecoes, organizacao e recursos de linguagem
- try/catch/finally/throw
- multiplos arquivos
- import
- null safety
- parametros opcionais e nomeados

## Ordem dos arquivos de exercicio

1. exercises/01_fundamentos_exercicios.dart
2. exercises/02_poo_basica_exercicios.dart
3. exercises/03_poo_avancada_exercicios.dart
4. exercises/04_excecoes_null_imports_exercicios.dart

## Como executar

No terminal do projeto:

```bash
dart run exercises/01_fundamentos_exercicios.dart
dart run exercises/02_poo_basica_exercicios.dart
dart run exercises/03_poo_avancada_exercicios.dart
dart run exercises/04_excecoes_null_imports_exercicios.dart
```

## Estrategia didatica

- Primeiro execute o arquivo completo para os alunos verem a saida.
- Depois comente partes do codigo e peca para os alunos completarem os TODOs.
- Reaproveite o mesmo dominio: Usuario, Produto, Pedido, Veiculo, ContaBancaria, Biblioteca, Livro.

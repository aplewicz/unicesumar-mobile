# Trilha Dart

Esta pasta concentra os conteúdos de Dart utilizados na disciplina.

## Objetivo da Trilha

Desenvolver a base de programação necessária para evoluir para Flutter, com foco em:

- Fundamentos da linguagem
- Programação orientada a objetos
- Exceções, null safety e organização de código
- Leitura e implementação de exercícios

## Estrutura

```text
dart/
  dart_app/
    bin/         # ponto de entrada para execução
    lib/         # classes e serviços reutilizáveis
    exercises/   # exercícios por tópico
    test/        # testes automatizados
```

## Projeto Principal

Local: `dart/dart_app`

Comandos básicos:

```bash
cd dart/dart_app
dart pub get
dart run bin/dart_examples.dart
dart test
```

## Roteiro de Estudo Sugerido

1. Execute os exemplos em `bin/` e analise as classes em `lib/`.
2. Resolva os arquivos da pasta `exercises/` na ordem numérica.
3. Compare sua implementação com o código discutido em aula.
4. Execute os testes para validar o comportamento.

## Entregas e Práticas

- Mantenha commits pequenos por exercício.
- Use nomes claros em classes, métodos e variáveis.
- Registre dúvidas no próprio código com comentários objetivos, quando necessário.

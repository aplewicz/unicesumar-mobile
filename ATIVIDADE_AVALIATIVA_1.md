# Trabalho 1 - Configuração do Ambiente e Hello World em Dart

**Valor: 0,5 ponto**

## Objetivo

Verificar se os estudantes conseguiram configurar corretamente o ambiente de desenvolvimento e executar um programa simples utilizando a linguagem **Dart**.

## Descrição da Atividade

O estudante deverá criar um projeto simples em Dart contendo um programa que imprima a mensagem **"Hello World"** no terminal.

Esta atividade tem como objetivo garantir que os estudantes consigam:

- configurar o ambiente de desenvolvimento
- executar programas Dart
- utilizar **Git e GitHub**
- documentar um projeto simples.

## Requisitos

O projeto deve conter:

- um arquivo `.dart`
- uma função `main()` que imprima uma mensagem no terminal.

Exemplo esperado de saída:

```text
Hello World
```

## Documentação (README)

O arquivo **README.md** deve conter:

- nome completo do estudante
- descrição do objetivo da atividade
- instruções para executar o programa
- exemplo da saída do programa no terminal.

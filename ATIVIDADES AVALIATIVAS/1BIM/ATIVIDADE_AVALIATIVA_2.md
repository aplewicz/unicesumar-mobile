# Trabalho 2 - CRUD de Biblioteca em Dart utilizando Orientação a Objetos

**Valor: 2,5 pontos**

## Objetivo

Aplicar conceitos de **Programação Orientada a Objetos em Dart** para desenvolver um sistema simples de gerenciamento de biblioteca executado no terminal.

## Descrição da Atividade

O grupo deverá desenvolver um programa em Dart que permita realizar operações básicas de gerenciamento de livros em uma biblioteca.

O sistema deverá funcionar por meio do **terminal**, permitindo ao usuário interagir com o programa utilizando um menu.

## Funcionalidades Mínimas

O sistema deve permitir:

- cadastrar um livro
- listar todos os livros cadastrados
- atualizar informações de um livro
- remover um livro

Essas operações caracterizam um **CRUD (Create, Read, Update, Delete)**.

## Requisitos Técnicos

O projeto deve obrigatoriamente utilizar:

- classes
- atributos
- métodos
- construtores
- listas para armazenar os objetos.

Sugestão de atributos para a classe `Livro`:

- título
- autor
- ano de publicação
- identificador (ISBN ou ID).

## Exemplo de Menu

```text
1 - Cadastrar livro
2 - Listar livros
3 - Atualizar livro
4 - Remover livro
5 - Sair
```
